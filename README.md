# 🍲 ระบบแนะนำเมนูอาหาร (Food Menu Recommendation System)

โปรเจคจบนี้เป็นการพัฒนาระบบเว็บแอปพลิเคชันสำหรับแนะนำเมนูอาหาร โดยมีการใช้โมเดล AI ในการช่วยประมวลผลและแนะนำทางเลือกที่เหมาะสมให้กับผู้ใช้งาน พร้อมระบบจัดการข้อมูลสูตรอาหารผ่านฐานข้อมูล

## 👨‍💻 คณะผู้จัดทำ
- นายอนุวรรตน์ เอนก
- นางสาวมินตรา สุขสมปรารถนา

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Backend:** Python (รวมถึงไฟล์ประมวลผลหลัก `app.py`, `blueprints/`)
- **Frontend:** HTML, CSS, JavaScript (โฟลเดอร์ `templates/` และ `static/`)
- **Database:** SQL (`recipes_db.sql` และ `db_utils.py`)
- **AI/ML:** โค้ดสำหรับการแนะนำประมวลผลเมนูอาหาร (`ai_model.py`)

## 📂 โครงสร้างโฟลเดอร์และไฟล์สำคัญ (Project Structure)
```text

 ┣ 📂 templates/       # ไฟล์ HTML สำหรับแสดงผลหน้าเว็บ
 ┣ 📜 app.py           # ไฟล์รันเซิร์ฟเวอร์หลักของโปรเจค
